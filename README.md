## üõ†Ô∏è Installation (Developer Mode)

Since this is a custom extension, you can install it manually in Chrome:

1.  Clone this repository:
    ```bash
    git clone https://github.com/BuildwithMc/pizzadaoos.git
    ```
2.  Open Chrome and navigate to `chrome://extensions`.
3.  Enable **Developer mode** (toggle in the top right).
4.  Click **Load unpacked**.
5.  Select the `pizzadao-os` folder from this repository.
6.  Open a new tab and enjoy **PizzaDAO OS**!
